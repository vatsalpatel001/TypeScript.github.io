→ TypeScript is a programming language.

→ Typescript is a superset of JavaScript.


→ Typescript code needs to be compiled while JavaScript code doesn't need to compile.


→ Typescript supports a feature of prototyping while JavaScript doesn't support this feature.

Why is TypeScript a superset of JavaScript?


→Typescript is a superset in the sense that you can completely write valid JavaScript without using any extra features, provide static type-checking
